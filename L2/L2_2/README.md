## test

```go
func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}
```
Порядок выполнения:
1. x = 1
2. return в качестве возвращаемого значения возьмет переменную x 
3. Выполняется defer, который увеличивает значение x на 1
4. выполнение функции завершится с результатом 2


## anotherTest
```go
func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}
```
Порядок выполнения:
1. x = 1
2. return в качестве возвращаемого значения создаст новую переменную в памяти и скопирует
в нее значение x = 1
3. Выполняется defer, который увеличивает значение x на 1, но не меняет значение переменной, которую возвращает return
4. выполнение функции завершится с результатом 1
