```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()
	for n := range ch {
		println(n)
	}
}
```
Программа выведет числа от 0 до 9, после чего произойдет Deadlock
и программа завершится с паникой. Это происходит из-за того,
что main (одна из горутин) вечно ожидает сообщений из открытого канала, который забыли закрыть
