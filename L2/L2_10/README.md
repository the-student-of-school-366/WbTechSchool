Для запуска программы сортировки соберите программу с помощью команды
```
go build main.go
```

после выполните в консоли одну из команду
```
# Простая сортировка из файла
.\main.exe test.txt

# Сортировка в обратном порядке
.\main.exe -r test.txt

# Только уникальные строки
.\main.exe -u test.txt
```
